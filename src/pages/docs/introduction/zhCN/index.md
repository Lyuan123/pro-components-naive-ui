# Pro Naive UI

Pro Naive UI 是基于 Naive UI 二次封装的。

要了解如何安装,参见 [安装](installation)。

它帮你解决了很多零碎、复杂的问题,也完全支持 Naive UI 的功能

## 使用 TypeScript

Pro Naive UI 全量使用 TypeScript 编写，支持完整的类型推导，和你的 TypeScript 项目无缝衔接。

## 快

我尽力让它不要太慢,至少内部没有编写任何有损于性能的代码
